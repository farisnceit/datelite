# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Vanilla DatePicker
- Single, range, and multiple date selection modes
- Comprehensive date blocking options
- Icon customization with position control
- Month/year dropdown navigation
- Range confirmation with Apply/Cancel buttons
- Custom positioning and append-to options
- 50+ CSS custom properties for theming
- TypeScript definitions
- Comprehensive test suite
- Full accessibility support
- Mobile-responsive design
- Keyboard navigation
- Locale support structure
- Time selection (24hr/AM-PM with seconds)
- Inline and popup display modes
- Method chaining API
- Event callback system
- No external dependencies

### Features
- **Selection Modes**: Single date, date ranges, multiple dates
- **Date Blocking**: Disable specific dates, ranges, weekdays, weekends, past/future dates
- **Customization**: 50+ CSS variables, custom themes, icon control
- **Navigation**: Month/year dropdowns, keyboard navigation
- **Positioning**: Custom positioning, append to any element
- **Time Support**: Optional time selection with seconds
- **Accessibility**: ARIA labels, keyboard support, screen reader friendly
- **Responsive**: Mobile-optimized interface
- **Framework Agnostic**: Works with any framework or vanilla JavaScript

### Browser Support

- Chrome Mobile 60+
