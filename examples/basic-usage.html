Basic DatePicker Usage Examples

Simple examples to get you started with the DatePicker component.

1. Single Date Selection

No date selected
JavaScript: const picker = new DatePicker('#single-date', { mode: 'single', format: 'Y-m-d', onChange: function(date) { console.log('Selected:', date); } });

2. Date Range Selection

No range selected
JavaScript: const rangePicker = new DatePicker('#range-date', { mode: 'range', format: 'Y-m-d', onChange: function(dateRange) { if (dateRange.start && dateRange.end) { console.log('Range:', dateRange.start, 'to', dateRange.end); } } });

3. Multiple Date Selection

No dates selected
JavaScript: const multiPicker = new DatePicker('#multiple-dates', { mode: 'multiple', closeOnSelect: false, onChange: function(dates) { console.log('Selected dates:', dates); } });

4. Date with Time Selection

No date/time selected
JavaScript: const datetimePicker = new DatePicker('#datetime', { mode: 'single', enableTime: true, time_24hr: false, format: 'Y-m-d h:i A', onChange: function(date) { console.log('Selected:', date); } });

5. Inline Calendar

No date selected
JavaScript: const inlinePicker = new DatePicker('#inline-calendar', { mode: 'single', inline: true, format: 'Y-m-d', onChange: function(date) { console.log('Selected:', date); } });
