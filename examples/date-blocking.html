Date Blocking Examples

Examples showing different ways to block or restrict date selection.

Block Weekends

No date selected
const picker = new DatePicker('#no-weekends', { disableWeekends: true, onChange: function(date) { console.log('Business day:', date); } });

Block Past Dates

No date selected
const picker = new DatePicker('#future-only', { blockPastDates: true, onChange: function(date) { console.log('Future date:', date); } });

Block Specific Dates

No date selected
const holidays = [ '2024-12-25', // Christmas '2024-01-01', // New Year '2024-07-04' // Independence Day ]; const picker = new DatePicker('#no-holidays', { disabledDates: holidays, onChange: function(date) { console.log('Non-holiday:', date); } });

Block Date Ranges

No date selected
const picker = new DatePicker('#no-vacation', { disabledDateRanges: [ { start: '2024-12-20', end: '2024-12-31' }, // Winter break { start: '2024-07-01', end: '2024-07-15' } // Summer vacation ] });

Only Specific Dates Allowed

No date selected
const allowedDates = [ '2024-12-01', '2024-12-05', '2024-12-10', '2024-12-15', '2024-12-20' ]; const picker = new DatePicker('#whitelist', { enabledDates: allowedDates });

Custom Block Function

No date selected
const picker = new DatePicker('#custom-block', { disableFunction: function(date) { // Block all Fridays return date.getDay() === 5; }, onChange: function(date) { console.log('Non-Friday:', date); } });

Dynamic Date Blocking Control

Use buttons to control date blocking
// Dynamic control methods picker.setDisableWeekends(true); picker.setBlockPastDates(true); picker.addDisabledDates(['2024-12-25']); picker.clearDisabledDates();
