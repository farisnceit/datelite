Getting Started

DateLite is a modern, lightweight, and highly customizable date picker component built with vanilla JavaScript.

Installation

Via NPM

npm install datelite

Via CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/datelite/dist/datelite.css"> <script src="https://cdn.jsdelivr.net/npm/datelite/dist/datelite.js"></script>

Manual Download

Download the files and include them in your project:

<link rel="stylesheet" href="path/to/datelite.css"> <script src="path/to/datelite.js"></script>

Basic Usage

Create a simple date picker with just a few lines of code:

<input type="text" id="datepicker" placeholder="Select a date"> <script> const picker = new DatePicker('#datepicker', { mode: 'single', format: 'Y-m-d' }); </script>

Configuration Options

Mode Options

Option Type Default Description
mode string 'single' Selection mode: 'single', 'range', or 'multiple'
inline boolean false Display calendar inline (always visible)
enableTime boolean false Enable time selection
time_24hr boolean true Use 24-hour time format
const picker = new DatePicker('#input', { mode: 'range', // Select date range enableTime: true, // Include time selection time_24hr: false // Use 12-hour format });

Display Options

Option Type Default Description
format string 'Y-m-d' Date format (Y-m-d, m/d/Y, etc.)
theme string 'default' Theme: 'default', 'dark', 'minimal'
showWeekNumbers boolean false Display week numbers
closeOnSelect boolean true Close calendar after selection

Position Options

Option Type Default Description
position string 'auto' 'auto', 'top', 'bottom', 'left', 'right'
appendTo string/element 'body' Element to append calendar to
positionX number null Fixed X position (pixels from left)
positionY number null Fixed Y position (pixels from top)
offsetX number 0 X offset from calculated position
offsetY number 0 Y offset from calculated position

Date Restrictions

Option Type Default Description
minDate Date/string null Minimum selectable date
maxDate Date/string null Maximum selectable date
disableWeekends boolean false Disable weekend dates
blockPastDates boolean false Disable past dates
disabledDates Array [] Array of specific dates to disable
enabledDates Array [] Only allow specific dates
disableFunction Function null Custom function to disable dates
const picker = new DatePicker('#input', { minDate: new Date(), // Today onwards maxDate: '2024-12-31', // Until end of year disableWeekends: true, // No weekends disabledDates: [ // Specific dates '2024-12-25', '2024-01-01' ], disableFunction: (date) => { // Custom logic return date.getDay() === 5; // Disable Fridays } });

Methods

open()

Opens the date picker calendar.

picker.open();

close()

Closes the date picker calendar.

picker.close();

toggle()

Toggles the calendar open/closed state.

picker.toggle();

setDate(date)

Programmatically set the selected date.

picker.setDate(new Date()); picker.setDate('2024-12-25');

getDate()

Get the currently selected date(s).

const selectedDate = picker.getDate();

clear()

Clear the selected date(s).

picker.clear();

destroy()

Destroy the date picker instance and remove all event listeners.

picker.destroy();

setDisableWeekends(boolean)

Dynamically enable/disable weekends.

picker.setDisableWeekends(true);

setBlockPastDates(boolean)

Dynamically enable/disable past dates.

picker.setBlockPastDates(true);

addDisabledDates(dates)

Add dates to the disabled dates list.

picker.addDisabledDates(['2024-12-25', '2024-01-01']);

clearDisabledDates()

Clear all disabled dates.

picker.clearDisabledDates();

Event Hooks

onReady

Called when the date picker is initialized and ready.

const picker = new DatePicker('#input', { onReady: function() { console.log('DatePicker is ready!'); } });

onChange

Called when a date is selected or changed.

const picker = new DatePicker('#input', { onChange: function(selectedDate) { console.log('Selected:', selectedDate); } });

onOpen

Called when the calendar is opened.

const picker = new DatePicker('#input', { onOpen: function() { console.log('Calendar opened'); } });

onClose

Called when the calendar is closed.

const picker = new DatePicker('#input', { onClose: function() { console.log('Calendar closed'); } });

onMonthChange

Called when the displayed month changes.

const picker = new DatePicker('#input', { onMonthChange: function(year, month) { console.log('Month changed to:', year, month); } });

onYearChange

Called when the displayed year changes.

const picker = new DatePicker('#input', { onYearChange: function(year) { console.log('Year changed to:', year); } });

Positioning Guide

Auto Positioning (Default)

The calendar automatically positions itself to stay within the viewport.

const picker = new DatePicker('#input', { position: 'auto' // Default behavior });

Fixed Position

Position the calendar at specific coordinates on the page.

const picker = new DatePicker('#input', { positionX: 100, // 100px from left positionY: 200 // 200px from top });

Relative Positioning with Offset

Adjust the position relative to the input element.

const picker = new DatePicker('#input', { position: 'bottom', offsetX: 10, // 10px right offset offsetY: 5 // 5px down offset });

Append to Custom Element

Append the calendar to a specific container instead of body.

const picker = new DatePicker('#input', { appendTo: '#custom-container' });

Position Options

position: 'left'

position: 'right'

Complete Examples

Date Range Picker

const rangePicker = new DatePicker('#date-range', { mode: 'range', format: 'Y-m-d', onChange: function(dateRange) { if (dateRange.start && dateRange.end) { console.log('From:', dateRange.start); console.log('To:', dateRange.end); } } });

Date & Time Picker

const datetimePicker = new DatePicker('#datetime', { mode: 'single', enableTime: true, time_24hr: false, format: 'Y-m-d h:i A' });

Inline Calendar

const inlinePicker = new DatePicker('#inline-calendar', { inline: true, mode: 'single' });

Restricted Dates

const restrictedPicker = new DatePicker('#restricted', { minDate: new Date(), maxDate: new Date(new Date().setDate(new Date().getDate() + 30)), disableWeekends: true, disabledDates: ['2024-12-25', '2024-01-01'] });
